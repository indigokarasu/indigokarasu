Indigo Karasu's public projects span agent runtimes, skills, automation, research, and utilities.

## Agent Runtime
- [hermes-agent](https://github.com/indigokarasu/hermes-agent), agent runtime: gateway, personas, tool orchestration

## Skills
- [forge](https://github.com/indigokarasu/forge), design and build agent skills
- [finch](https://github.com/indigokarasu/finch), mine session history for behavioral patterns
- [praxis](https://github.com/indigokarasu/praxis), measure skill outcomes against baselines
- [skilllab](https://github.com/indigokarasu/skilllab), audit, merge, prune, and publish skills

## Automation
- [spot](https://github.com/indigokarasu/spot), appointment booking across SevenRooms, Resy, Tock, OpenTable
- [reach](https://github.com/indigokarasu/reach), real-time data: prices, weather, news
- [sands](https://github.com/indigokarasu/sands), natural-language calendar management
- [voyage](https://github.com/indigokarasu/voyage), trip planning and itinerary construction

## Intelligence
- [scout](https://github.com/indigokarasu/scout), structured OSINT research with provenance
- [sift](https://github.com/indigokarasu/sift), web research synthesis and source reliability scoring
- [weave](https://github.com/indigokarasu/weave), provenance-backed social graph and contacts
- [taste](https://github.com/indigokarasu/taste), personalized recommendations from consumption signals

## Operations
- [custodian](https://github.com/indigokarasu/custodian), autonomous platform monitoring and auto-repair
- [genie](https://github.com/indigokarasu/genie), VPS disk cleanup and backup retention
- [bower](https://github.com/indigokarasu/bower), automatic Google Drive organizer
- [lucid](https://github.com/indigokarasu/lucid), nightly journal curation into the memory store

- [mcp-plu-upc](https://github.com/indigokarasu/mcp-plu-upc), UPC/barcode and PLU produce lookups

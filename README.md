**indigokarasu** is a personal intelligence agent — a single identity that grows its own capabilities, remembers what it learns, and acts across the tools you already use.

Built on [Hermes Agent](https://github.com/indigokarasu/hermes-agent), OCAS (Open Cognitive Agent Suite) is the skill layer: authored capabilities for research, memory, calendar, dispatch, automation, and self-improvement. Every module is a named skill. Every skill is versioned, auditable, and composable.

## Core

- [hermes-agent](hermes-agent) — the runtime: gateway, persona system, tool orchestration
- [indigo](indigo) — the Indigo Karasu profile: identity, memory, context, and day-to-day operation
